# ***Математическая модель Промышленного кластера*** 
## Сокращения 
ПК - Промышленный кластер,  
УК  - Участник ПК,  
УКи - Участники ПК,  
ЯПК - Ядро ПК
## Метематическая модель 
$$S=\langle C,\overline{C},P,R,\overline{R}, \overline{\overline{R}}\rangle$$ -  система ПК, где  
$$C=\{c\_1, ..., c\_n\}$$ - УКи,  
$$\overline{C}\subset C$$-ЯПК
